# Contributing to Clone-Wars

Thank you for your interest in contributing to Clone-Wars! This document provides guidelines and instructions for contributing.

## Code of Conduct

Please read and follow our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) before contributing.

## How to Contribute

### Reporting Issues

If you find a broken link or a clone that no longer works:

1. Check if the issue already exists in [Issues](https://github.com/fkyoureshit-cloud/Clone-Wars/issues)
2. If not, create a new issue with:
   - The clone name and link
   - Description of the problem
   - Screenshot if applicable

### Adding New Clones

Before submitting a new clone, ensure:

- ✅ It's a clone/alternative of a **popular software or app**
- ✅ Project has **minimal functionality** (not just UI-only)
- ✅ **No duplicates** of existing clones (unless different tech stack)
- ✅ Links are **working and verified**
- ✅ Proper **alphabetical order** is maintained

#### Submission Process

1. **Fork** the repository
2. **Create a branch**: `git checkout -b add/clone-name`
3. **Edit** [README.md](README.md) with your new clone entry
4. **Follow the table format**:
   - Clone/Alt of | Demo | Tutorial | GitHub | Tech Stack
5. **Commit**: `git commit -m "Add [Clone Name] clone"`
6. **Push**: `git push origin add/clone-name`
7. **Create a Pull Request** with description of the clone

### Fixing Existing Entries

1. Fork the repository
2. Create a branch: `git checkout -b fix/clone-name`
3. Update the broken or incorrect entry in README.md
4. Commit and push your changes
5. Create a Pull Request explaining what was fixed

### Updating Documentation

Help improve our documentation:

- Improve README clarity
- Fix typos or formatting
- Add missing information
- Create guides or tutorials

## PR Review Process

All PRs will be reviewed for:

- ✅ Relevance and quality of clone
- ✅ Accuracy of links
- ✅ Proper formatting
- ✅ Alphabetical order
- ✅ No duplicates

## Questions?

Feel free to open an issue for questions or suggestions. We appreciate all contributions!
